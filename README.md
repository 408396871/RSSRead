#RSSRead
RSSRead是一个iOS设备上的RSS/Atom阅读器。 

##已有主要功能
* Add a new feed
* Offline feed read
* Star the content you like
* Mark as read and mark all as read
* 能够删除管理已添加rss源

* [MBProgressHUD](https://github.com/jdg/MBProgressHUD)
