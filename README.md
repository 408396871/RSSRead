#RSSRead
RSSRead是一个iOS设备上的RSS/Atom阅读器。

##在App Store上已上架，应用中文名叫“已阅”  
<https://itunes.apple.com/us/app/yi-yue-rss-li-xian-xin-wen-yue-du/id850246364?ls=1&mt=8>  

##已有功能  
* 添加rss地址  
* 本地离线rss内容  
* 收藏  
* 标记已读和标记全部已读  
* 能够删除管理已添加rss源  

##使用的技术   
* 解析rss用的是[MWFeedParser](https://github.com/mwaterfall/MWFeedParser)  
* 本地存储CoreData
